# 문제 이해하기 
N 회원 1 ~ N
i 회원이 들 수 있는 역기 무게 Wi 회원들 사이에 M개의 친분
Aj번 회원과 Bj번 회원이 친분
이 헬스장에서 자신이 최고라고 생각하는 회원 수 ?

# input
5 3 		// N M	// 회원, 친분
7 5 7 7 1	// W1 W2 W3 W4 W5
1 2		// Aj Bj
2 3		
3 4

# output
2

조건
2 ≤ N ≤ 105
Aj ≠ Bj
