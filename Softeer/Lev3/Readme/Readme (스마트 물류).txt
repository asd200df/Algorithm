// 스마트 물류 문제 설계

# 1
1) input 받기 
	받을 때 빈 배열 하나 더 맹글어서 로봇 위치에 1 채워 넣기 
2) 로봇 위치를 vector에 저장 
3) vector를 vec.size로 for문 돌기
	돌면서 K위치에 0있으면 2로 바꾸고 1이거나 2면 continue
4) 다 돌면 2 갯수 세서 최대값 갱신 해주면 될 것 같은데
+ 왼쪽부터 2 채워야 함 

# 2
1) input 받기 
	받을 때 빈 배열 하나 더 맹글어서 로봇 위치에 1 채워 넣기 
2) for문 돌기
	1이나 2 그리고 3이면 continue
	0이면 K 범위에 1(로봇)이 있는지 봐야 돼 있으면 0을 2로 바꾸고
	1을 3으로 바꿔 (사용한 로봇이라는 표시) 
3) 다 돌면 2 갯수 세서 최대값 갱신 해주면 될 것 같은데
+ 그리디 (최대한 많이)

# 3
1) input 받기 
	받을 때 빈 배열 하나 더 맹글어서 로봇 위치에 1 채워 넣기 
2) for문 돌기
	1이나 2 그리고 3이면 continue
	0이면 isValid 함수로 idx 보내주고	
	isValid 함수에서 K 범위에 1(로봇) 있는지 확인 후 있으면 0 >> 2로
	1 >> 3으로 바꿔줌
3) 다 돌면 2 갯수 세서 최대값 갱신 해주면 될 것 같은데
+ 그리디 (최대한 많이)


idx = 2 K = 2
0 1 2 3 4 5 idx
- - 0 - -
2 - 2 = 0
2 + 2 = 4
0 ~ 4
