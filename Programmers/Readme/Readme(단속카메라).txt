#1 
1 진입 낮은 순서대로(오름차순) Sort 하기 
2 카메라 위치 제일 낮은 위치 (-21e8) 저장해놓기 (초기설정)
3 현재 카메라 위치보다 더 크면 카메라 설치(Count++) && 현재 카메라 위치 갱신
	(카메라 위치는 현재 구간의 진출 위치로 넣어야 함)
4 반복 !
5 결과 출력
